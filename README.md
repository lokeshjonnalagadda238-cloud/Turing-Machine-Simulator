# Turing-Machine-Simulator
Turing Machine Simulator

A Turing Machine Simulator is a program that mimics the working of a Turing Machine, a fundamental concept in computer science introduced by Alan Turing. This project helps visualize how computations are performed step by step on an infinite tape.

 Features
Simulates a single-tape deterministic Turing Machine
Step-by-step execution of transitions
Custom input tape support
Displays tape movement (Left/Right)
Accept and Reject state detection
Simple and user-friendly interface
 How It Works

The simulator uses the following components:

States (Q) – Set of machine states
Input Alphabet (Σ) – Symbols given as input
Tape Alphabet (Γ) – Symbols allowed on tape
Transition Function (δ) – Defines rules
Start State – Beginning of computation
Accept/Reject States – Final states
Execution Steps:
Read symbol under the tape head
Apply transition rule
Write new symbol
Move head (Left/Right)
Change state
